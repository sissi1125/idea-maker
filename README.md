# Marketing RAG Playground

一个可调试的 RAG 驱动运营 idea 生成 Playground。

用户上传产品文档，或从已上传文档库中选择历史 document version 后，可以检查每个 RAG pipeline 阶段，检索可信证据，并生成带引用的产品画像、卖点地图和可发布内容 idea。

当前仓库包含 Marketing RAG Playground 的 harness 工程基座，以及阶段 1 的 RAG Playground 拆分说明。开始前运行：

```bash
./init.sh
```

然后阅读：

- `AGENTS.md`
- `docs/PRODUCT.md`
- `docs/ARCHITECTURE.md`
- `docs/RAG_PIPELINE_PLAYGROUND.md`
- `feature_list.json`
- `progress.md`
